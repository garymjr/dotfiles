---
description: Commit changes using jj (Jujutsu version control)
---

Use jj to commit changes. Follow these steps:

1. First, run `jj status` to see the current state
2. Run `jj diff` to review what changes will be committed
3. Run `jj log --limit 5` to see recent commit history for context
4. Finally, run `jj commit` to commit the current changes

When committing, use a clear, descriptive commit message that follows the existing commit message style in the repository.

If there are any issues or if you need to make adjustments before committing, address them first before running the final commit command.
